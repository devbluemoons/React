# React
  
self-study & Ref.  
  
###### ref. `리액트를 다루는 기술`  
  
###### install `yarn`
```
npm install --global yarn
```
  
###### check `yarn version`
```
yarn --verion
```
  
###### install `create-react-app`
```
yarn global add create-react-app
```
  
###### create `react web application`
```
create-react-app [project-name]
```
  
###### run `react web application`
```
yarn start
```
